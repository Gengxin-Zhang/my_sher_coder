#### 俺对MarkDown的理解

MarkDown是一种纯文本格式的标记语言。<sub>（html是超文本标记语言）</sub>。
对于暑假期间学过html的我来说，学 MarkDown比一般人更容易，学了 MarkDown之后我才发现， MarkDown与html是两个境界。
MarkDown实在是太方便了，大大节约了写作时间。它可以让作者摆脱排版的困扰，专心写作。
